# Full stack Authentication

## Getting Started

First, initialize the shadcn sdk:

```bash
pnpm dlx shadcn-ui@latest init
```

Then, run the development server:

```bash
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
